# Raw Data

This folder contains the original airline operational dataset used in this project.

The dataset was obtained from the U.S. Department of Transportation On-Time Performance database.

Due to file size limitations, the raw data file is not included in this repository.

The raw dataset includes:

- Flight schedule information
- Arrival and departure delays
- Aircraft identifiers (Tail Number)
- Distance and airport metadata
- Delay breakdown categories

No modifications were made to the raw file prior to cleaning.
