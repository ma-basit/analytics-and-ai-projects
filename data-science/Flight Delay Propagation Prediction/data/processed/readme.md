# Processed Data

This folder contains cleaned and model-ready datasets generated during the project.

Due to file size limitations, processed datasets are not included in this repository.

The following files are generated through the notebooks:

- `flight_data_clean.parquet`
- `flight_data_with_target.parquet`
- `flight_data_model_ready.parquet`
- `flight_data_model_ready_v2.parquet`

Each notebook produces intermediate datasets used in later stages.

These processed files include:
- Cleaned operational records
- Defined delay propagation target
- Engineered features for modeling
