# Processed Data

This folder contains cleaned and transformed datasets generated during the project.

These files are not included in the repository to keep the project lightweight.

## Contents

- Cleaned order-level dataset
- Feature-engineered dataset used for modeling

## Notes

- Data in this folder is created from the raw dataset through preprocessing steps
- Includes aggregated data, cleaned columns, and engineered features

## How to Recreate

Run the notebooks in order:

1. Data Understanding & Cleaning
2. Exploratory Data Analysis
3. Feature Engineering

These steps will generate the processed datasets required for modeling.
