# Analytics and AI Projects

## About This Repository
This repository contains a curated collection of projects across **Data Analytics**, **Data Science & Machine Learning**, and **AI Systems**.

The focus is on:
- Realistic problem statements  
- Industry-aligned workflows  
- Clear project structure and documentation  

Each project is self-contained in its own folder with a dedicated README.

---

## Notes on Data and Dashboards
- Raw datasets are **not included** due to size and licensing constraints  
- Tableau and Power BI packaged files are **not uploaded**  
- Dashboard screenshots and insights are provided instead  
- Each project README documents:
  - Dataset source  
  - Tools used  
  - Analysis workflow  
  - Key insights
